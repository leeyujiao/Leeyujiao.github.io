Scanner类：
功能：可以实现键盘输入数据，到程序中。

引用类型的一般使用步骤：

1.导包：
import 包路径.类名称;
如果需要使用的目标类，和当前类位于同一个包下，则可以省略导包语句不写；
只有Java.lang包下的内容不需要导包。
lang包提供基础类，最重要的是Object（它是类层次结构的根）和 Class（它的实例表示正在运行的应用程序中的类）

2.创建：
类名称 对象名 =new 类名称()；

3.使用：
对象名.成员方法()

import java.util.Scanner;//导包
public class Demo01Scanner {

    public static void main(String[] args) {
        //创建
        //System.in代表从键盘输入
        Scanner in=new Scanner(System.in);

         //使用
        //获取键盘输入的一个int数字： int num=in.nextInt();
        //获取键盘输入的一个字符串：String str=in.next();
        int num=in.nextInt();
        String str=in.next();

        System.out.println(num);
        System.out.println(str);
    }
}

匿名对象：
只有右边的对象，没有左边的名字和赋值运算符。
例如：
new 类名称（）；
注意事项：
匿名对象只能使用唯一的一次，下次再用不得不再创建一个新对象；
使用建议：
如果确定有一对象只需要使用唯一的一次，就可以使用匿名对象；

public class Demo01Anonymous {
    public static void main(String[] args) {
        new Person().showName("lyjiao");
    }
}



public class Person {

    public void showName(String name){
        System.out.println("我叫:"+name );
    }
}

Random类：
用来生成随机数字，使用的三个步骤：
导包：
import java.util.Random
创建：
Random r=new Random();//小括号留空
使用：
获取一个随机的数字(范围是int的所有范围，有正负两种)：int num=r.nextInt();
获取一个随机的数字(参数代表范围，左闭右开区间) int num=t.next(3);
实际代表的含义是：[0,3),也就是0~2

ArrayList类：

数组有一个缺点，一旦创建，程序运行期间长度不可以发生改变，故用集合来弥补其缺陷。

//数组实现：
    Person[] arr=new Person[3];

    Person one=new Person("李四",23);
    Person tow=new Person("王五",21);
    Person three=new Person("赵四",33);
    //将one中的地址值赋值到数组的0号元素位置
    arr[0]=one;
    arr[1]=tow;
    arr[2]=three;

    System.out.println(arr[1].getAge()+arr[1].getName());
}

数组的长度不可以发生改变，
但ArrayList集合的长度是可以随意变化的。

对于ArrayList来说，有一个代表泛型
泛型：也就是装在集合中的所有元素，全部是统一的类型
注：泛型只能是引用类型，不能是基本类型
public static void main(String[] args) {
//创建一个ArrayList集合，集合的名称是arraylist，里面装的全都是String类型的数据
//<>中必须是引用类型
//备注：从JDK1.7开始，右侧的尖括号内部可以不写内容，但是<>本身不能省略

    ArrayList<String >  arrayList=new ArrayList<>();

    System.out.println(arrayList);//[]

    //向集合中添加元素需要用到add（）方法
    arrayList.add("李四");
    System.out.println(arrayList);//[李四]

    arrayList.add("路飞");
    System.out.println(arrayList);//[李四,路飞]
}

ArrayList当时中的常用方法：

public boolean add(E e):向集合中添加元素，元素的类型和泛型一致，返回值代表是否成功
备注：对于ArrayList集合来说，add添加动作一定是成功的，所以返回值可用可不用
但是对于其他集合来说，add添加动作不一定成功
public E get(int index):从集合中获取元素，参数是索引编号，返回值是集合元素
public E remove(int index):从集合中删除元素，参数是索引编号，返回值是删除的元素
public int size():获取集合的尺寸长度，返回值是集合中包含元素的个数

public static void main(String[] args) {
    ArrayList<String > list=new ArrayList<>();
    //向集合中添加元素
    boolean success = list.add("索隆");

    System.out.println(list);//[索隆]
    System.out.println(success);//true

    list.add("路飞");
    list.add("山治");
    list.add("黑胡子");
    list.add("乌索普");
    System.out.println(list);//[索隆, 路飞, 山治, 黑胡子, 乌索普]
    //从集合中获取元素

    String s = list.get(2);
    System.out.println(s);//山治

    //从集合中删除元素

    String remove = list.remove(3);
    System.out.println("删除的是"+remove);//删除的是黑胡子
    System.out.println(list);//[索隆, 路飞, 山治, 乌索普]

    //测试集合的元素个数
    int size = list.size();
    System.out.println(size);//4
}

为什么<>里不能写基本类型：
因为集合里存入的是地址值，而基本数据类型没有地址值
怎么用集合储存基本数据类型元素
使用基本类型对应的“包装类”

基本类型            包装类
byte                    Byte
short                   Short
int                     Integer       【**】
double              Double
float                   Float
long                    Long
char                    Charecter       【**】
boolean             Bloolean

从JDK1.5开始，支持自动装箱，自动拆箱
自动装箱：基本类型--->包装类型
自动拆箱：包装类型--->基本类型
